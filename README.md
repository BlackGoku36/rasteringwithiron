# rasteringwithiron
Basic rasterizer written in Haxe with help of Iron Library for learning purpose.

# Features
* PBR.
* Cel-Shade(little joke, need to do more).

# Movements
* WASD to move around.
* Q to go strafe and E to strafe up.
* Mouse click to lock mouse.
* Escape to unlock mouse.
